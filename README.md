## Run Locally

Clone the project

```bash
  git clone https://github.com/chadirabii/advice-2-you
```

Go to the project directory

```bash
  cd advice-2-you
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn run dev
```
